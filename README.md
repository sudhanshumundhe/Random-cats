# 🐱 Random Cat Viewer

A modern and visually appealing React application that displays random cat profiles using an API. The app features a clean UI, soft animated backgrounds, and interactive elements to enhance user experience.

---

## 🚀 Live Demo

https://catrandomviewer.netlify.app/

---

## 📸 Features

* 🐱 Fetch random cat profiles from API
* 🖼️ Display cat image with breed details
* 🎨 Modern UI with gradient background and 3D card design
* ✨ Smooth animations and hover effects
* 🔄 Generate new cat with a single click
* 📱 Responsive layout (works on different screen sizes)

---

## 🔌 API Used

Random Cat API:
https://api.freeapi.app/api/v1/public/cats/cat/random

## ▶️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/cat-viewer.git
cd cat-viewer
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```
